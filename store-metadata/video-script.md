## ISTRUZIONI REGISTRAZIONE (Windows)

### Step 1 — Installa gli strumenti
- **Android Studio** (emulatore) → https://developer.android.com/studio
- **OBS Studio** (registrazione schermo) → https://obsproject.com — GRATIS
- **CapCut** (montaggio) → https://www.capcut.com/it — GRATIS

### Step 2 — Configura l'emulatore
1. Apri Android Studio → Device Manager → crea Pixel 8 (API 34)
2. Imposta lingua italiana nell'emulatore
3. Installa l'app con `npx expo start --android`

### Step 3 — Registra con OBS
1. Apri OBS → Aggiungi sorgente "Acquisizione schermo" → seleziona la finestra emulatore
2. Imposta output: 1080×1920 (o taglia dopo), 60fps, MP4
3. Registra OGNI scena separatamente (più facile da montare)
4. Usa shortcut `Ctrl+Shift+R` per iniziare/fermare registrazione

### Step 4 — Monta in CapCut
1. Importa tutti i clip
2. Aggiungi sottotitoli animati (testo stile "typewriter" o "pop")
3. Usa musica royalty-free da CapCut library (categoria: "uplifting" o "motivational")
4. Aggiungi transizioni morbide tra le scene (0.3 secondi)
5. Esporta: 1080×1920, 30fps, MP4
